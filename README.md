# Number Guessing Game

This project is a Java-based console application developed as part of the Oasis Infobyte Java Development Internship.

## Project Description
The Number Guessing Game is an interactive Java application where the player tries to guess a randomly generated number within a limited number of attempts. The game also includes multiple rounds and a scoring system.

## Features
- Random Number Generation
- Multiple Game Rounds
- Limited Attempts
- Score Calculation System
- User Interaction through Console
- Replay Option

## Technologies Used
- Java
- Random Class
- Scanner Class
- Loops and Conditional Statements

## How to Run
1. Open the project in any Java IDE.
2. Compile the program.
3. Run the `NumberGuessingGame.java` file.
4. Start guessing the number and play the game.

## Game Rules
- Guess a number between 1 and 100.
- You have 7 attempts in each round.
- Points are awarded based on remaining attempts.
